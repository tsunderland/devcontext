Get started in 2 minutes

1

Install DevContext

One command to install. Uses pipx for clean, isolated install.

pipx install devcontext
2

Initialize your project

Navigate to any project and start tracking.

cd ~/my-project && devctx init
3

Work normally

Start sessions, add notes, end when done. DevContext captures the rest.

devctx start && devctx note "working on auth"
4

Resume anytime

Come back days later and instantly know where you left off.

devctx resume
